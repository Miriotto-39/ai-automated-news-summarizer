# AI Automated News Summarizer (AI自動ニュース要約・レポート生成ツール)

## 概要
指定したニュースソースから最新のトピックを自動的に取得し、AI（OpenAI GPT-4o-mini）がビジネスや社会への影響度を判断して要約レポートを生成・保存するツールです。

## 主な機能
- **自動スクレイピング**: Googleニュース等のプラットフォームから最新のニュース見出しを抽出。
- **AIインテリジェンス**: 大量のニュースから重要なトピックを3つ厳選し、簡潔に要約。
- **スケジュール実行**: 1日3回（朝・昼・晩）など、設定した時間に完全無人で動作。
- **自動レポート保存**: 実行時のタイムスタンプ付きでテキストファイルを生成。

## 使用技術
- **Language**: Python 3.12
- **Framework**: FastAPI
- **AI API**: OpenAI API (Model: gpt-4o-mini)
- **Library**: BeautifulSoup4 (Scraping), Schedule (Automation), Requests

## プロジェクトの背景
「情報収集の効率化」をテーマに開発しました。毎日手動で行っていたニュースチェックを自動化し、数秒で今日の重要事項を把握できる仕組みを構築。APIコストを最小限に抑えつつ、最大限の情報を得るためのフィルタリングロジックを実装しています。
