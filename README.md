# 📰 AI Automated News Summarizer（AIニュース自動要約ツール）

> Googleニュースから最新情報を自動取得し、AIが重要トピックを厳選・要約して保存するツールです。

---

## 📌 このツールで解決できること

- 毎日大量のニュースを読む時間をなくしたい
- 重要なニュースだけをAIに選んでまとめてほしい
- 日付ごとにニュースレポートを自動保存・管理したい

---

## 🎬 デモ

**自動生成されるレポートの中身：**

```
【AI厳選：今日の重要ニュース】
1. **イギリス・フランスでの航行の自由確保会議**：英スターマー首相がアメリカのホルムズ海峡封鎖を支持しない意向を示し、国際的な海上輸送の安全性に影響を与える可能性がある。

2. **イラン、海上輸送の一時停止を検討**：和平交渉の進展を図る中で、イランが海上輸送の一時的な停止を検討しており、関係国のシーライン経済に影響を及ぼすことが懸念される。

3. **米中軍事的緊張の高まり**：米軍による海上封鎖に対して中国が抗議し、タンカーとの衝突リスクが高まっており、国際貿易やエネルギー供給に悪影響を及ぼす可能性がある。
```

ファイルは `news_report_実行日時.txt`（例：`news_report_20260415_0016.txt`）として自動保存されます。

## 🚀 セットアップ・使い方

### 1. リポジトリをクローン
```bash
git clone https://github.com/Miriotto-39/ai-automated-news-summarizer.git
cd ai-automated-news-summarizer
```

### 2. 必要なライブラリをインストール
```bash
pip install -r requirements.txt
```

### 3. 環境変数を設定
`.env` ファイルを作成し、以下を記入してください：
```
OPENAI_API_KEY=your_api_key_here
```

### 4. サーバーを起動
```bash
uvicorn main:app --reload
```

### 5. 稼働状況を確認
ブラウザまたはcurlで以下にアクセス：
```
http://localhost:8000/status
```

**レスポンス例：**
```json
{
  "status": "自動監視中",
  "info": "毎日08:00, 12:00, 19:00に自動実行されます。"
}
```

レポートは毎日 **08:00 / 12:00 / 19:00** に自動生成されます。手動実行は不要です。
